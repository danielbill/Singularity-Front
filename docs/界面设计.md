## UI

### 板块设计

- 快讯：简洁的电报快讯
- 爆点：关注近期最火爆的事物，比如clawdbot
- 人物：以传奇人物为线索，单独梳理出和他相关的新闻，按照时间线排列，呈现过去3个月的轨迹
- 企业：以奇点企业为核心，展现相关动态


### PC端设计

- 请使用frontend-design skill
- 页面构成 图片头 + 导航栏 + 内容区，尺寸结构请自行定义
- 导航栏：快讯 | 奇点 | 前线 | 异动 
- 首页 - 快讯 （站点url/news）
  - PC端分为左右两个区域
  - 左边为 奇点动态，当日和传奇人物相关的新闻都出现在这里
  - 右边为 前沿资讯，排除掉奇点动态的新闻在这里
  - 右边的新闻量会更大
